Cómo obtener el token
  1. Convertí tu cuenta IG a Business o Creator
  2. Vinculá la cuenta IG a una Página de Facebook
  3. Creá una App en developers.facebook.com (tipo Business)
  4. Agregá el producto Instagram Graph API
  5. Generá el token en Graph API Explorer con permisos: instagram_content_publish, pages_read_engagement, instagram_basic
  6. Convertí el token a long-lived (~60 días)
  7. Obtené el Instagram Business Account ID via Graph API
