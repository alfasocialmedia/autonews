¿Qué es AutoNews Connector?

AutoNews Connector es un plugin de WordPress que permite a tu sitio recibir y publicar artículos generados por el sistema AutoNews de forma automática, segura y sin configuraciones complejas.

Autenticación segura
Token único generado automáticamente. No requiere contraseñas de usuario.
Categorías automáticas
El plugin crea categorías y etiquetas si no existen en WordPress.
Imagen destacada
Descarga y asigna la imagen destacada al post automáticamente.
SEO integrado
Compatible con Yoast SEO y RankMath. Configura keywords y meta descripciones.

Requisitos

  • WordPress 5.8 o superior
  • PHP 7.4 o superior
  • Acceso de administrador al panel de WordPress
  • La API REST de WordPress habilitada (activa por defecto)

Instalación

1
Obtener el archivo del plugin

El archivo ZIP del plugin es provisto por el equipo de AutoNews. El archivo se llama autonews-connector.zip.

Si no tenés el archivo ZIP, solicitalo a tu agencia o proveedor de AutoNews.
2
Acceder al panel de WordPress

Ingresá a tu sitio con una cuenta de administrador:

https://tusitio.com/wp-admin
3
Ir a la sección de plugins

En el menú lateral izquierdo, hacé clic en:

Plugins → Añadir nuevo → Subir plugin
4
Subir el archivo ZIP

Hacé clic en Elegir archivo, seleccioná el archivo autonews-connector.zip y luego hacé clic en Instalar ahora.

5
Activar el plugin

Una vez instalado, WordPress mostrará un mensaje de éxito. Hacé clic en Activar plugin.

El plugin instalado se llama "AutoNews Connector" y aparecerá en la lista de plugins activos.

Configuración

6
Abrir la página de configuración del plugin

En el menú lateral de WordPress, hacé clic en:

Ajustes → AutoNews
7
Copiar los datos de conexión

En la página de configuración verás un panel con los datos necesarios para conectar AutoNews con este sitio. Encontrarás:

  • URL del sitio — la dirección de tu WordPress
  • Token de autenticación — clave única generada automáticamente
  • Datos de conexión JSON — para copiar y pegar directamente en AutoNews

Ejemplo de cómo se verá el token:

autonews_tk_4f9a2b8c1e3d7f6a0b5c9d2e4f8a1b3c
Importante: Tratá el token como una contraseña. No lo compartas públicamente. Si se compromete, podés regenerarlo desde esta misma página.

Vincular con AutoNews

8
Acceder al panel de AutoNews

Ingresá al sistema AutoNews con tu usuario y contraseña.

Configuración → WordPress
9
Agregar o editar el sitio WordPress

Hacé clic en Agregar sitio (si es la primera vez) o en el botón Editar del sitio existente.

Completá los campos:

  • Nombre: nombre interno para identificar el sitio
  • URL del sitio: la dirección completa (ej: https://tusitio.com)
  • Token del plugin: pegá el token copiado desde Ajustes → AutoNews en WordPress
10
Guardar la configuración

Hacé clic en Guardar. El sitio aparecerá en la lista con el badge Plugin indicando que usa el conector.

Verificar la conexión

11
Usar el botón "Probar Plugin"

En la lista de sitios WordPress en AutoNews, cada sitio configurado con el plugin tiene un botón Probar Plugin.

Al hacer clic verás uno de estos mensajes:

Éxito: "Plugin conectado — NombreSitio (WP 6.x)" → todo funcionando correctamente.
Error "Token inválido": el token fue copiado incorrectamente. Repetí el paso 7.

Solución de problemas

  • Verificá que la URL del sitio en AutoNews sea correcta y no tenga barra al final.
  • Confirmá que el plugin esté activado en WordPress (Plugins → Plugins instalados).
  • Verificá que la API REST de WordPress no esté bloqueada por un plugin de seguridad.
  • Accedé a https://tusitio.com/wp-json/autonews/v1/status desde el navegador — debería devolver {"ok":false,"error":"Unauthorized"}. Si no carga, la API REST está bloqueada.

  • Asegurate de estar ingresado con una cuenta de Administrador en WordPress.
  • Desactivá y volvé a activar el plugin desde Plugins → Plugins instalados.

  • Verificá que WordPress tenga permisos de escritura en la carpeta wp-content/uploads/.
  • Si la imagen viene de una URL externa, asegurate de que WordPress pueda hacer peticiones HTTP salientes (función wp_remote_get habilitada).

  1. En WordPress: Ajustes → AutoNews → Regenerar token.
  2. Copiá el nuevo token.
  3. En AutoNews: Configuración → WordPress → Editar sitio y pegá el nuevo token.

Los artículos ya publicados permanecerán en WordPress sin cambios. AutoNews dejará de poder publicar nuevos artículos hasta que se reinstale el plugin o se configure la conexión por REST API estándar.
